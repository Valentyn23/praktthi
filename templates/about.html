Про додаток

Опис проекту

Це демонстраційний веб-додаток, створений в рамках практичної роботи з розробки Telegram-ботів з використанням бібліотеки Aiogram 3.0.

Функціонал

  • Telegram бот - підбір систем відеоспостереження за параметрами
  • База даних SQLite - зберігання користувачів, систем та замовлень
  • Імітована оплата - система балансів та поповнення
  • Веб-інтерфейс - керування завданнями (To-Do List)
  • Аутентифікація - реєстрація та авторизація користувачів
  • CRUD операції - створення, читання, оновлення та видалення завдань

Технології

Backend
  • Python 3.11+
  • Aiogram 3.0
  • Flask
  • SQLAlchemy
Frontend
  • HTML5
  • CSS3
  • Bootstrap 5
  • Bootstrap Icons
Примітка: Дані зберігаються в локальній базі даних SQLite. Система оплати є імітованою для навчальних цілей.
